      # NoneBot是什么？

      如果你使用过 NoneBot ，可马上进入插件使用说明。
      如果你不了解 NoneBot ，你可以参考 从零开始 的安装教程。

      如果你是开发者，你还可以学习：

      cqhttp 项目地址： cqhttp github

      nonebot 开发文档： 开发文档

      # 可能用到的

      如果你使用过 NoneBot ，非常建议您先安装 nonebot 的计时器基础依赖，这在很多 NoneBot 插件内都需要使用到：

          pip install "nonebot[scheduler]"
